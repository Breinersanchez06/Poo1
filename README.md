EJERCICIO 1: USUARIOS
Sistema escolar con herencia y polimorfismo. Usuario es la clase base (nombre, id, password). Administrador, Profesor y Estudiante heredan y sobrescriben leerNota() con acciones propias. Main crea instancias y muestra comportamientos diferenciados.
EJERCICIO 2: ZOOLOGICO
Modelado de animales con herencia y polimorfismo. Animal (abstracta) define atributos comunes y métodos abstractos. Mamífero y Reptil agregan características. Subclases (León, Elefante, Serpiente, Cocodrilo) implementan comportamientos únicos. main usa arreglo polimórfico para ejecutar acciones uniformemente.
